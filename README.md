
# PANTHEON Intelligence OS

**Live Global Awareness Dashboard** · v14 · Live Demo: [https://anthonydanielonline-ui.github.io/pantheon-intel/](https://anthonydanielonline-ui.github.io/pantheon-intel/)

PANTHEON is a self-contained HTML-based situational awareness workstation for analysts, operators, and decision-makers needing real-time global intel at a glance. Deployable anywhere—no backend required.

## 🚀 Key Features

- **🕸️ Intelligence Network Graph**: Interactive nodes (conflict, sanctions, maritime, etc.) sized by threat severity; drag, zoom, click for details. Solid/dashed links show direct/inferred relationships.
- **🎯 Threat Radar**: Live threats categorized by time horizon (immediate 0–24h, near-term 1–7d, emerging 7–30d). Hover for details, click to drill down.
- **📡 Live Intelligence Terminal**: Streaming feeds for breaking news, cyber, markets, infra. Analyst notes and customizable filters.
- **⏱️ Timeline Replay**: Scrub through last 24h/7d/30d/90d across conflict/cyber/market events.
- **👥 Entity Tracker**: Sort/filter flights, vessels, conflicts, cyber by severity/speed/name. Live updates.
- **🌍 Globe View**: Day/night modes with overlays for cables, IADS, cams, GPS, fire, weather, human activity.
- **Fully Responsive**: Works on desktop, tablet, mobile. Pure HTML/CSS/JS—no dependencies.

## 🎯 Who it's for

- Geopolitical analysts, risk officers, ops centers
- Traders monitoring markets/infra/cyber risks
- Security teams tracking vessels/flights/conflicts
- Anyone needing a customizable "common operational picture"

## 📦 Quick Setup

1. Download `index.html` + assets from this repo.
2. Open `index.html` in any modern browser.
3. Configure data sources via the built-in stubs (JSON endpoints for feeds/graph/timeline).
4. Deploy to any static host (GitHub Pages, Netlify, etc.).

**Production tips**: Add your own API keys for live feeds (e.g., news APIs, ship trackers). Customize modules via simple JS config.
